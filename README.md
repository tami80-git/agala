# 🚼 אתגר העגלה: הגרסה התל אביבית

משחק פיקסל ארט בסגנון רטרו על אמא עם עגלה שמנסה לשרוד את מכשולי המדרכה של תל אביב!

## 🎮 תיאור המשחק

ברוכים הבאים ל"אתגר העגלה" - משחק פלטפורמר קפצני בסגנון פיקסל ארט שמדמה את החוויה האמיתית של הליכה עם עגלת תינוק ברחובות תל אביב-יפו!

### ✨ פיצ'רים

- 🏙️ **סקייליין של תל אביב** - כולל מרכז עזריאלי, מגדלים מודרניים ובניינים בסגנון באוהאוס
- 🎨 **גרפיקה בסגנון רטרו** - פיקסל ארט מפורט וצבעוני
- 🚼 **עגלת תינוק מפורטת** - עיצוב מודרני עם 4 גלגלים וכיסוי
- 🏛️ **לוגו העירייה** - המעגל הצבעוני האייקוני של תל אביב-יפו
- 🎯 **מערכת קומבו** - צברו נקודות בונוס על ידי הימנעות ממכשולים ברצף
- 📱 **תמיכה במובייל** - כפתורי מגע למשחק בנייד

### 🚧 מכשולים

נווטו בין המכשולים האמיתיים של תל אביב:

- 🗑️ **פחי אשפה** - ירוקים, כתומים וכחולים עם לוגו העירייה
- 💩 **קקי כלבים** - עם זבובים מעופפים!
- 🚲 **אופניים שוכבות** - חונות על המדרכה
- 🌳 **גזם ענפים** - ערמות של קשקושי גיזום
- 🛴 **קורקינטים** - קורקינטים חשמליים נטושים
- 🦅 **ציפורים** - עפות בגבהים שונים

### 🎮 בקרה

**מחשב:**
- ⬆️ **רווח** - קפיצה
- ⬇️ **חץ למטה** - כריעה

**מובייל:**
- 👆 **הקש** - קפיצה
- 🤚 **החזק** - כריעה

### 🏆 מערכת ניקוד

- נקודות בסיס עבור כל מכשול שעוברים
- **קומבו x5**: +10 נקודות בונוס
- **קומבו x10**: +20 נקודות בונוס
- **קומבו x15+**: +50 נקודות בונוס!

המהירות עולה ככל שמתקדמים במשחק 🔥

## 🚀 איך לשחק

### אופציה 1: שחקו באינטרנט (GitHub Pages)

פשוט היכנסו ל: `https://[YOUR-USERNAME].github.io/tlv-stroller-game`

### אופציה 2: הורידו ושחקו מקומית

1. הורידו את הקובץ `index.html`
2. פתחו אותו בדפדפן
3. לחצו START ותתחילו לשחק!
